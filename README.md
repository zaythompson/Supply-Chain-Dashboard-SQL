# Supply Chain Performance Dashboard-SQL

## Overview

This project showcases an interactive Power BI dashboard designed to analyze supply chain performance across procurement, inventory, supplier quality, and logistics operations. Built using PostgreSQL, SQL, Power BI, and DAX, the dashboard provides decision-makers with actionable insights into delivery performance, supplier quality, forecasting accuracy, procurement activity, and operational efficiency.

## Business Problem

Organizations rely on accurate supply chain data to monitor supplier performance, procurement efficiency, inventory movement, and logistics operations. Without centralized reporting, it becomes difficult to identify operational bottlenecks, evaluate supplier performance, and make informed purchasing decisions. This dashboard provides executive-level visibility into key supply chain KPIs to support data-driven operational improvements.

---

## Project Objectives

- Monitor executive supply chain KPIs
- Analyze procurement and inventory performance
- Evaluate supplier quality metrics
- Compare delivery performance across delivery modes
- Monitor forecast accuracy
- Support data-driven operational decision-making

---

## Key Insights

- Forecast Accuracy averaged **80.10%**, indicating relatively consistent demand forecasting.
- Air delivery recorded the highest average order volume.
- Supplier Quality Scores remained above **5.3** across all delivery modes.
- Procurement activity was distributed fairly evenly across delivery methods.
- Average Lead Time Variance averaged approximately **2.5 days**, highlighting opportunities to reduce delivery variability.

---

## Business Recommendations

- Improve forecasting processes to further reduce inventory risk.
- Continue monitoring supplier quality to maintain operational performance.
- Investigate delivery methods with higher lead time variance to improve consistency.
- Optimize procurement planning using forecasting insights.
- Monitor order volume by delivery mode to improve logistics planning and resource allocation.
